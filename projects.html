    Wireless Sensor Network Optimization by using FreeRTOS for ESP32 Microcontroller

    Supervisor : En HASRULNIZAM BIN HASHIM

    Executive Summary :

    The ESP32 Microcontroller can produce a Wireless Sensor Network using its own communication, which is ESPNOW, a standard 2.4GHz wireless communication protocol, however this performance and range are greatly limited, restricting its WSN potential. The paper suggests that such a limitation be addressed through using FreeRTOS to enable deep sleep mode and dual core processing power with task scheduler features, including by adding an additional transmitter to serve as a middle man between the sender and receiver.

    Wireless Sensor Network Wildfire Early Warning System Using Applied based on FDRS Malaysia

    Supervisor : En HASRULNIZAM BIN HASHIM

    Executive Summary :

    Wildfire Can be devastating not only by destroying the forest but also by causing casualties. The project proposes a system integration solution with IoT for tackling wildfires, with the system serving as an early warning system. The system will monitor the forest environment and send alerts to the appropriate authorities if any incidents occur. The Malaysia Fire Danger Rating System is used as a standard in measuring the level of wildfire and detecting potential scenarios.
